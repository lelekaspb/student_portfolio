On this student portfolio website, you will find some of the assignments from the first semester.

The first semester was divided into four themes: Basic Web, Basic UX, Basic Animation, and Basic Content.

basic web theme

In this theme, we were given a basic but intensive introduction to primary tools of the Multimedia Design profession.
Here we learned to create a website from scratch that works on the platforms from mobile to desktop.

basic web assignments

basic UX theme

In this theme, we experienced a selection of UX methods and learned how to present our products and communicate our research and test results to stakeholders.
The theme gave us a basic understanding of the interactions between users and user interfaces, as well as the theories, tools, and tests of developing a digital product.

basic UX assignments

basic animation theme

In this theme, we learned how to build more complex interactive interfaces with CSS animations and JavaScript.
We learned more about visual design to engage the user, and techniques for idea-generation, prototyping, style-developing, as well as other production management tools in order to handle more complex projects.

basic animation assignments

basic content theme

This theme predominantly focused on video as the content type.
We learned how to plan, shoot, edit, and present a video at a basic professional level and implement it on a responsive website based on a brief from a company.
